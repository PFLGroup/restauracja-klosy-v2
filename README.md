# Restauracja Kłosy — landing v2 (KONCEPT)

> Kuchnia polska z sercem. Wesela · komunie · catering. Warszawa, Ochota.
> **Koncepcyjna wersja v2** w kinowym, ciemnym stylu. Realizacja: **PFL Group** ([pflgroup.pl](https://www.pflgroup.pl/)).

Jednostronicowy, statyczny landing (`HTML + CSS + JS`, bez frameworków) — alternatywna,
„kinowa" odsłona strony Restauracji Kłosy. Hostowany na **GitHub Pages**.

🔗 **Podgląd na żywo:** https://pflgroup.github.io/restauracja-klosy-v2/

---

## ℹ️ Status

- **Charakter:** koncept / propozycja v2 (oznaczone na stronie plakietką „Koncept v2").
- **Indeksowanie:** wyłączone (`noindex` + `robots.txt`) — nie konkuruje z produkcyjną stroną Kłosów.
- **Repozytorium:** osobne, publiczne (wymóg GitHub Pages na planie Free). W repo trafia tylko
  zbudowany, zminifikowany wynik — czytelne źródła (rozdzielone CSS/JS) trzymane poza repo.
- **Formularz „Zapytaj o termin":** tryb demo (nic nie wysyła). Aby zbierać zapytania, podepnij
  endpoint Formspree/Tally w źródle i przebuduj.
